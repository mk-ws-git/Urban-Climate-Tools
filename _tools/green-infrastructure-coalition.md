## Overview

The Green Infrastructure Coalition provides comprehensive tools and resources to help communities plan, design, and implement green infrastructure solutions. Their platform supports sustainable stormwater management and urban greening initiatives.

### Key Features

- **Planning Guides**: Step-by-step implementation frameworks
- **Design Templates**: Standardized green infrastructure designs
- **Cost-Benefit Calculator**: Estimate project costs and benefits
- **Best Practice Library**: Successful case studies and lessons learned
- **Community Engagement Toolkit**: Resources for public participation

## Use Cases

- Municipal stormwater management planning
- Community greening projects
- Sustainable development review
- Grant application support
- Public education and outreach
