## Overview

ArcGIS Urban is a comprehensive web-based 3D application designed for urban planners to support planning and decision-making workflows. It enables scenario planning, zoning analysis, and impact assessment through powerful visualization tools.

### Key Features

- **3D Visualization**: Create immersive 3D models of urban environments
- **Scenario Planning**: Compare multiple development scenarios
- **Zoning Analysis**: Understand zoning impacts on development potential
- **Impact Assessment**: Evaluate environmental and social impacts
- **Stakeholder Engagement**: Share plans with the public via web apps

## Use Cases

- Comprehensive plan development
- Zoning code analysis and reform
- Development review and permitting
- Community engagement and visualization
- Climate adaptation scenario planning

## Getting Started

1. Visit [ArcGIS Urban](https://www.esri.com/en-us/arcgis/products/arcgis-urban)
2. Request a demo or trial
3. Work with Esri to configure your urban model
4. Import local GIS data and zoning information
