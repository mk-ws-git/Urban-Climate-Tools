## Overview

The EPA National Stormwater Calculator is a desktop application that estimates the annual amount of rainwater and frequency of runoff from a specific site. Users can evaluate how different types of green infrastructure practices can reduce runoff.

### Key Features

- **Site Analysis**: Estimate runoff for any location in the US
- **Green Infrastructure Scenarios**: Model rain gardens, permeable pavement, green roofs, and more
- **Climate Projections**: Include future climate scenarios in analysis
- **Cost Estimation**: Basic cost-benefit analysis for practices
- **Report Generation**: Create documentation for projects

## Use Cases

- Site development planning
- Green infrastructure sizing
- MS4 permit compliance
- Low impact development design
- Watershed planning

## Getting Started

1. Visit [EPA Stormwater Calculator](https://www.epa.gov/water-research/stormwater-calculator)
2. Download and install the desktop application
3. Enter your site location and characteristics
4. Model green infrastructure scenarios
5. Review runoff reduction estimates
