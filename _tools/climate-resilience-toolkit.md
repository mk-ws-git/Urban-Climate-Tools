## Overview

The Climate Resilience Toolkit provides resources, tools, and case studies to help communities plan for and adapt to climate change impacts. Developed by the Georgetown Climate Center, this comprehensive platform supports local governments and organizations in building resilience.

### Key Features

- **Steps to Resilience**: Guided framework for resilience planning
- **Climate Explorer**: Access local climate data and projections
- **Case Study Library**: Learn from successful adaptation projects
- **Tool Finder**: Discover relevant tools and resources
- **Training Modules**: Build capacity for resilience planning

## Use Cases

- Climate adaptation planning
- Hazard mitigation planning
- Community resilience assessments
- Grant writing and project development
- Stakeholder engagement and education

## Getting Started

1. Visit [Climate Resilience Toolkit](https://climateresiliencetoolkit.org)
2. Use the Steps to Resilience framework
3. Explore the Climate Explorer for local data
4. Review relevant case studies and tools
