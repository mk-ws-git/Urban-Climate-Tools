## Overview

i-Tree is a suite of free software tools from the USDA Forest Service that helps communities assess and manage urban forests and community trees. These peer-reviewed tools provide urban and rural forestry analysis and benefits assessment.

### Key Features

- **i-Tree Eco**: Comprehensive analysis of urban forest structure, environmental effects, and value
- **i-Tree Canopy**: Estimate tree cover using aerial imagery
- **i-Tree Design**: Visualize tree benefits for individual properties
- **i-Tree Landscape**: Assess ecosystem services at landscape scale
- **i-Tree Species**: Help select appropriate tree species for specific locations

## Use Cases

- Urban forest inventory and assessment
- Carbon sequestration quantification
- Air quality improvement estimation
- Stormwater management analysis
- Tree planting program planning

## Getting Started

1. Visit [i-Tree Tools](https://www.itreetools.org)
